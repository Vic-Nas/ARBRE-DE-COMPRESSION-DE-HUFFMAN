## Installation

1. Clone the repository:

```bash
git clone https://github.com/Vic-Nas/PythonSolutions.git
cd PythonSolutions
````

2. Install dependencies:

```bash
pip install -r requirements.txt
```

## Usage

Run the main script:

```bash
python main.py
```

## Requirements

* Python 3.x
* Any libraries listed in `requirements.txt`
